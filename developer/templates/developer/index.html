Overview

This page provides an overview of the BC&YI platform.

The platform consists of a configurable 3-tier system, including the following layers:

  • Data Layer
  • Business Layer
  • Presentation Layer
  • Configuration Layer

    • The Data layer is build as a SQL database using MySQL. The Business Layer is a web server built using Python3 and Django. The Presentation Layer is built using JavaScript other web technologies that are all served from the Business Layer.

    The Configuration layer is built and maintained using the other three layers. The Configuration layer describes the types of objects that the system models. The data for the Configuration layer is contained within the Data layer.

    The entire system is served using Amazon Web Services.
