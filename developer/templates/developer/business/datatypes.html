Data Types

This page describes all of the data types for values of objects in the system.

The list of data types is identified as enumerator values of the "_data type" _uuname.

Data TypeDescription
_object

A value that represents another object.

The JSON representation of this field is an object that contains two fields: an "id" field that contains the unique descriptor of the instance which is referenced by this value and a "description" field that contains a human readable version of that instance.

_string A value of up to 255 unicode characters.

The JSON representation of this field is a string.

_number A whole or real number.

The JSON representation of this field is a string.

_datestamp A date in the calendar. The date is stored internally as a string in the following format: "yyyy-mm-dd" where yyyy is the year, mm is the number of the month (01-12) and dd is the number of the day (01-31). This format conforms with ISO 8601 date format.

The JSON representation of this field is a string in ISO 8601 date format.
