Path Examples

Following are examples of paths constructed using different forms.

Javascript code to create a path to all of the Experience objects that contain the specified user.

var path = "#" + userInstance.getValueID() + '::reference(Experience)';

Javascript code to create a path to all of the Service objects whose name is housing.

path = "Service>_name[_text="Housing"]::reference(Service)"

Javascript code to create a path to all of the _user objects whose email address starts with the string 'val'.

path = "_user[_email^="+val+"]"
