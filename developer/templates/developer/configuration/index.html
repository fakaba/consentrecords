Overview

This page provides an overview of the configurations of information in the BC&YI platform.

The configuration layer is stored in as data in the system that is used to identify the objects that are modeled in the system.

Configuration terms whose name begins with an underscore ("_") are reserved by the system for bootstrapping the configuration of other elements.
